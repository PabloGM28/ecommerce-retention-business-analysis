# E-commerce Retention & Growth Business Analysis (Portfolio)

## ES — Resumen
Proyecto de portfolio orientado a **Business Analytics**. El objetivo es diagnosticar por qué un e-commerce de moda (no lujo) puede estar **creciendo más lento de lo esperado**, usando datos públicos y herramientas estándar en empresa.

**Stack:** BigQuery (SQL), Excel, Tableau, R  
**Enfoque:** hipótesis de negocio → métricas → análisis → recomendaciones accionables.

## EN — Summary
Business-oriented portfolio project to diagnose why a non-luxury fashion e-commerce may be **growing slower than expected**, using public data and standard analytics tools.

**Stack:** BigQuery (SQL), Excel, Tableau, R  
**Approach:** business hypotheses → metrics → analysis → actionable recommendations.
